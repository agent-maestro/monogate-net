# Monogate Network Navigator

Static MVP for `monogate.net`.

This is an ecosystem navigator, not a command center or dashboard. It links the
main Monogate surfaces and marks private/internal links clearly.

## Files

- `index.html`
- `styles.css`

## Local Review

Open `index.html` in a browser. No build step, network access, server, auth, or
package install is required.

Optional local server:

```bash
python -m http.server 8080
```

Then open:

```text
http://localhost:8080
```
